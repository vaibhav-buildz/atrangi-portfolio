# Atrangi Portfolio

Welcome to my 3D Interactive Portfolio!

This portfolio showcases my professional journey, projects, and skills in a unique, interactive experience featuring a 3D pirate-themed map layout.

## Features

*   **Interactive 3D Map:** Explore my career journey through an engaging 3D map interface.
*   **Terminal Simulation:** A sleek terminal interface for viewing skills and information.
*   **Dynamic Animations:** Smooth transitions and engaging CSS/JS animations.

## Technologies Used

*   HTML5
*   CSS3 (Custom styling, variables, animations)
*   JavaScript (DOM manipulation, event handling)
*   3D libraries/techniques

## Local Development

To run this project locally:

1.  Clone this repository.
2.  Serve the directory using a local web server. For example: `npx serve . -l 3000`
3.  Open `http://localhost:3000` in your web browser.
